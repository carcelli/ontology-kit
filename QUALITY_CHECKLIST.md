# Quality Assurance Checklist Template
## Before AI Code Generation:
- [ ] Clear problem statement written
- [ ] Expected inputs/outputs defined
- [ ] Edge cases identified
- [ ] Performance requirements noted

## After AI Code Generation:
- [ ] Code runs without syntax errors
- [ ] Logic matches requirements
- [ ] Edge cases handled
- [ ] Performance acceptable

## Quality Gates (Automated):
- [ ] make quality (lint + format + typecheck)
- [ ] make test (unit tests pass)
- [ ] make dryrun (pre-deployment validation)

## Learning Verification:
- [ ] Can explain code logic in plain English
- [ ] Understands key algorithms/data structures
- [ ] Knows how to modify for edge cases
- [ ] Can debug common issues
